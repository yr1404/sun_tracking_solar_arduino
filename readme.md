# Project Sunflower
This is a **<i>sun tracking solar panel</i>** which tracks sun's current position and moves panel's face towards it.  
  
Parts used in this project:
- Arduino Uno
- Breadboard
- LDR
- Servo motor
- Solar Panel and LED
- Few jumper wires and resistors
